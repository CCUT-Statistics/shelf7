菜单项设置详解

ID
一般情况不需要动。
排序
数字越小越靠前，越大越靠后；建议以10的倍数作为顺序，方便后期添加菜单项时将菜单项插入到希望出现的位置中。
图标
输入Font Awesome图标名称，或图标图片的网址，不填写则不显示图标。
导航标签
显示在页面上的文字。
网址
输入网址或“魔法菜单项”；外链使用https://或https://前缀，内链可以去掉本站网址前缀（如https://example.net/thread-1234.htm → thread-1234.htm）。
Title属性
鼠标悬停在上面时会看到的说明文字。
描述
某些插件会用到此项，作为说明文字。
CSS类
给该菜单项添加CSS类，实现一些效果。
自定义属性
给该菜单项添加自定义属性，实现一些效果。
二级菜单
选择一个菜单即可让该菜单项拥有二级菜单；若选择了二级菜单，则该菜单的链接会被忽略。
