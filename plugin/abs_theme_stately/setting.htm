Stately主题设置指南

为了帮助您更好地理解和使用本主题的设置，我精心准备了这份指南。以下是本主题设置的三个关键维度，引导您轻松完成设置过程。

一、要设置什么类别

首先，您需要明确要设置的类型，即第一个菜单条的内容。

本主题提供了丰富的设置类型，主要包括：

全局
涵盖站点的基本配置，如标志、联系方式等。
外观
调整站点的视觉效果，包括颜色、背景等。
外观-板式、外观-板式微调
定义不同页面和区域的布局和版式。
自定义内容、自定义内容-页面
允许用户添加或修改站点内的特定文本或信息。
特色功能
提供增强用户体验的额外功能，如快速发帖框、阅读模式等。
自定义代码
为高级用户提供添加自定义CSS和JavaScript代码的选项。
高级设置
针对站点性能和安全的高级配置选项。

二、要设置哪里

确定了要设置的类型后，接下来您需要明确这些设置将应用于哪些位置或场景，即第二个菜单条的内容。

主要包括：

全局
适用于整个站点的配置。（无此标记通常也是全局设置）
主页
顾名思义。
论坛板块页
展示论坛版块里的帖子列表的页面。同时也代表主页的帖子列表。
帖子详情页
展示帖子详细内容和回帖列表的页面。
用户
展示用户信息和个人中心的页面。
独立页面
其他独立的站点页面，包括论坛版块列表、发帖页面、登录、注册等。

三、设置具体选项

最后，您只需滚动到您想要调整的具体选项即可。

例如：

我想调整网站的颜色：
外观 → 颜色 → 配色（主色调）
我想更改欢迎辞内容：
自定义内容 → 主页-欢迎辞 → 欢迎辞内容
我想调整帖子列表的样式：
外观-版式 → 帖子列表 → 帖子列表风格（全局）
我想在帖子页面上显示上一篇和下一篇文章的导航链接：
外观-版式微调 → 帖子详情-通用 → 显示文章导航
我希望用户能在个人中心里设置自己喜欢的配色：
外观-版式微调 → 用户页面 → 显示“个性化”部分
我想更改登录页面的标题：
自定义内容-页面 → 登录 → 页面标题

通过这三个维度的详细指导，相信您能轻松完成主题设置，打造出一个符合您需求的个性化站点。
